## About

Azure consultant with a platform engineering and DevOps foundation, focused on the part of cloud work that is hard to reverse: landing zone design, identity, governance, and infrastructure as code. I deliver greenfield builds and on-premises migrations for enterprise clients, and I care more about decisions a team can operate and defend than about elegant diagrams nobody maintains.

My background is operational at scale. I have run hybrid estates of several hundred VMs and App Services in 24/7 environments, migrated business-critical workloads with zero downtime, and built IaC pipelines that keep infrastructure consistent and auditable. The architecture I like best is the one the client's team can still own on day 365.

I also work with AI workloads on Azure (Azure OpenAI in production, secure deployment patterns on AKS), and I enjoy teaching the fundamentals: my [SkyCraft](https://github.com/mbiszczanik/skycraft) project is a hands-on Azure learning path built from that interest.
